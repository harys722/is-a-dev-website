is-a.dev is a service that allows developers to get a sweet-looking .is-a.dev subdomain for their personal websites.
To register your own subdomain, go to the is-a-dev/register repository on GitHub and follow the instructions.

Report abuse

If you believe one of the subdomains is abusing the service, please report it.
